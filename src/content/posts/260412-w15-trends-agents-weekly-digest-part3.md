## 요약

### 2026-04-11 (1건)

- **OpenHands Software Agent SDK v1.16.1**: OpenHands는 2024년 OpenDevin 시절부터 "Docker 샌드박스 필수 + UI 포함 모놀리식 앱"으로 성장해왔다. 2026-04-02 v1.16.1과 함께 **기존 V0 아키텍처가 2026년 4월 완 [원문](https://github.com/OpenHands/software-agent-sdk)

### 2026-04-12 (5건)

- **Qwen Code**: **Qwen Code**는 "An open-source AI agent that lives in your terminal"을 표방하는 Alibaba의 오픈소스 AI 코딩 에이전트다. 2026년 4월 기준 22.8k★ [원문](https://github.com/QwenLM/qwen-code)
- **Multica**: 기존 AI 코딩 도구 사용 패턴은 **프롬프트 복사-붙여넣기 → 결과 확인 → 반복**이다. Multica는 이를 **태스크 할당 → 자율 실행 → 결과 보고**의 팀 협업 패턴으로 전환한다: [원문](https://github.com/multica-ai/multica)
- **Meta, 차기 AI 모델 오픈소스 버전 공개 예고**: - **출처**: Axios scoop (2026-04-06/07) [원문](FETCH_FAILED)
- **GitHub Agentic Workflows**: **출처**: GitHub Blog (2026-02-13 Technical Preview) [원문](FETCH_FAILED)
- **Archon**: 기존 AI 코딩 에이전트(Claude Code, Codex, Cline 등)는 프롬프트에 따라 실행 경로가 달라지는 비결정적(non-deterministic) 특성을 갖는다. Archon은 **YAML 워크플로우 정 [원문](https://github.com/coleam00/Archon)
