## 요약

### 2026-04-05 (1건)

- **Microsoft Agent Framework 1.0 GA**: Microsoft가 **Agent Framework 1.0 정식 버전(GA)**을 출시했다. Semantic Kernel의 엔터프라이즈 기반(세션 관리, 타입 안전성, 미들웨어, 텔레메트리)과 AutoGen의 혁신적 [원문](https://devblogs.microsoft.com/agent-framework/microsoft-agent-framework-version-1-0/)

### 2026-04-08 (1건)

- **MoCha ComfyUI 워크플로우 (WanVideoWrapper 통합)**: **대상:** Wan2.1-T2V-14B 백본 위에서 동작하는 단일 마스크 캐릭터 교체 워크플로우 [원문](https://github.com/Orange-3DV-Team/MoCha)

### 2026-04-11 (3건)

- **Google Colab MCP Server**: **Colab MCP Server**는 Google이 오픈소스로 공개한 MCP(Model Context Protocol) 서버로, 핵심 기능은 _로컬에서 돌아가는 AI 에이전트_와 _브라우저 Colab 세션_을 한  [원문](https://github.com/googlecolab/colab-mcp)
- **vLLM v0.19.0**: **vLLM v0.19.0**(2026-04-03 릴리즈)은 **448 커밋 / 197 기여자(신규 54명)**가 참여한 대형 릴리즈로, 이전 v0.18 시리즈에서 실험적으로 들어왔던 여러 하위 피처가 한 번에 안정 [원문](https://github.com/vllm-project/vllm/releases/tag/v0.19.0)
- **Rowboat**: **Rowboat**는 2026년 4월 초 기준 11.8k★ / 1.1k forks / 1,496 commits / TypeScript 96.7%로 성장 중인 오픈소스 "AI coworker" 프로젝트다. 자체 슬로 [원문](https://github.com/rowboatlabs/rowboat)
