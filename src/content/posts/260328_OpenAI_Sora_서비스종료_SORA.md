# OpenAI Sora 서비스 완전 종료 — 영상 생성 AI 시장 재편

> OpenAI가 2026년 3월 25일 Sora 앱과 API를 완전 종료. 출시 6개월 만에 높은 컴퓨트 비용과 사용자 감소로 서비스 중단 결정.

## 핵심 내용

OpenAI는 2026년 3월 25일 텍스트-투-비디오 플랫폼 Sora의 앱과 API를 모두 종료한다고 발표했다. 2025년 9월 공개 출시 이후 약 6개월 만이다.

종료 사유로 OpenAI는 "컴퓨트 수요 증가에 따른 우선순위 조정"을 언급했으며, Sora 연구팀은 로보틱스를 위한 월드 시뮬레이션 연구에 집중할 것이라고 밝혔다. 실질적으로는 생성 비디오당 추론 비용이 지속가능하지 않았던 것이 핵심 원인이다.

이에 따라 Disney와의 10억 달러 투자 딜도 무산되었다. Disney는 2025년 12월 Sora에서 자사 캐릭터를 활용한 사용자 생성 AI 비디오를 허용하는 계약을 체결했으나, OpenAI의 방향 전환으로 진행이 중단되었다.

사용자 지표 측면에서 2026년 초 분석 데이터에 따르면, Sora 앱은 2025년 11월 대비 12월에 신규 다운로드가 32% 감소하는 등 매월 연속 하락세를 보였다.

## 시사점

Sora의 실패는 AI 영상 생성 분야에서 세 가지 구조적 교훈을 남긴다. 첫째, 생성 비디오당 추론 비용이 수익 모델을 뒷받침하지 못하면 서비스 지속이 불가능하다. 둘째, 독립형 AI 영상 앱보다 기존 에디터(CapCut, Premiere Pro 등)에 통합되는 형태가 사용자 유지에 유리하다. 셋째, Disney 딜 무산은 AI 생성 콘텐츠의 IP 관리와 품질 보증이 엔터프라이즈 계약의 핵심 허들임을 보여준다.

## R&D 적용 가능성

Sora 종료는 영상 생성 AI 시장의 구조적 전환점이다. 단일 서비스에 의존하는 영상 파이프라인은 서비스 종료 리스크에 직접 노출된다. Memesis 프로젝트의 멀티 API 통합 노드 기반 아키텍처(SDXL, Flux, Runway 등 복수 백엔드)가 이러한 리스크를 분산하는 올바른 설계 방향임을 실증하는 사례다. 향후 Seedance 2.0, Kling 등 대안 서비스의 API 안정성과 비용 구조를 우선 평가할 필요가 있다.
