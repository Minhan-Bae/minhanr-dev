## 요약

### 2026-04-05 (3건)

- **GSOPs 2.9**: **개발:** David Rhodes, Ruben Diaz (CG Nomads) [원문](https://github.com/cgnomads/GSOPs)
- **HoudiniMCP**: HoudiniMCP는 Model Context Protocol(MCP)을 통해 Claude AI가 SideFX Houdini를 직접 제어할 수 있게 하는 오픈소스 통합 도구이다. 자연어 프롬프트로 노드 생성, 파라미 [원문](https://github.com/capoomgit/houdini-mcp)
- **LL3M**: LL3M은 대형 언어 모델(LLM) 팀을 활용하여 Blender용 Python 코드를 작성함으로써 3D 에셋을 생성하는 멀티에이전트 시스템이다. 기존의 3D 데이터셋 학습 방식과 달리, 3D 형상 생성을 **코드 작 [원문](https://arxiv.org/abs/2508.08228)

### 2026-04-06 (1건)

- **SceneVerse++**: 3D 장면 이해를 위한 주석 데이터는 희소하고 비용이 높은 반면, 인터넷에는 무한한 비주석 비디오가 존재한다. 본 논문은 **자동화된 데이터 엔진**으로 웹 큐레이션된 비주석 비디오를 고품질 3D 학습 데이터로 변환 [원문](https://arxiv.org/abs/2604.01907)

### 2026-04-08 (1건)

- **Houdini_ComfyUI**: AxioWorld(NXStorm·Caremba·Lilien86 공동개발)의 Houdini_ComfyUI는 Houdini 패키지로 설치되어 ComfyUI를 백그라운드에서 자동 기동하고, Houdini 노드 그래프 안에 [원문](https://github.com/AxioWorld/Houdini_ComfyUI)

### 2026-04-11 (3건)

- **Make-It-Poseable**: 3D 캐릭터를 포즈시키는 일(posing)은 CG/CV의 근본 연산이다. 전통적 파이프라인은 rigging → skinning → mesh vertex displacement 순으로 기하(geometry)를 직접 변 [원문](https://arxiv.org/abs/2512.16767)
